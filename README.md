# SystemVerilog-Processor-Design
Custom processor designed in SystemVerilog with integrated ALU, accumulator, instruction register, program counter, control unit, and memory interface for FPGA-oriented development and verification.

This repository contains a custom processor developed in SystemVerilog for FPGA-oriented design and simulation. The project includes datapath and control components such as the ALU, accumulator, instruction register, program counter, multiplexers, memory interface, and control logic.

The work highlights digital design, processor architecture fundamentals, hardware verification, and modular HDL development using tools such as Quartus and Questa.
