# SystemVerilog-Processor-Design
Custom processor designed in SystemVerilog with integrated ALU, accumulator, instruction register, program counter, control unit, and memory interface for FPGA-oriented development and verification.

This repository contains a custom SystemVerilog processor built for FPGA-oriented development and simulation. The design includes integrated datapath and control components such as an ALU, accumulator, instruction register, program counter, memory interface, and control unit. The project demonstrates digital logic design, processor integration, HDL verification, and computer architecture fundamentals.
